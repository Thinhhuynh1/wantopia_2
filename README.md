# Wantopia - Trạm Ước Vọng

Project được sắp xếp lại theo cấu trúc tập trung cho site nhiều trang:

- `front-end/`: toàn bộ các trang HTML nguồn
- `front-end/assets/css/styles.css`: style dùng chung cho site
- `front-end/assets/js/script.js`: menu mobile, reveal on scroll, form demo và chuyển trang
- `front-end/assets/img/`: logo và hình ảnh cho các page
- `front-end/assets/video/`: video hero

## Các trang chính

- `front-end/index.html`: trang chủ
- `front-end/su-kien.html`: trang sự kiện
- `front-end/cua-hang.html`: trang cửa hàng
- `front-end/tam-tu.html`: trang tâm tư
- `front-end/lien-he.html`: trang liên hệ
- `front-end/login.html`: trang đăng nhập demo

## Chạy dự án

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```
