# Wantopia - Trạm Ước Vọng

Project hiện được tổ chức lại thành nhiều trang HTML để dễ quản lý giao diện:

- `index.html`: trang chủ
- `su-kien.html`: trang sự kiện
- `cua-hang.html`: trang cửa hàng
- `tam-tu.html`: trang tâm tư
- `lien-he.html`: trang liên hệ
- `styles.css`: style dùng chung cho toàn bộ site
- `script.js`: menu mobile, reveal on scroll, form tâm tư và chuyển trang có hướng

## Chạy dự án

```bash
npm install
npm run dev
```

## Build production

```bash
npm run build
```
